# Version v0.0.12

This release improves the reliability of the web UI, updates several dependencies for better performance and security, and includes minor maintenance tasks.

## New Features

- No new user-facing features were added in this release.

## Bug Fixes

- Fixed an issue where the web UI could not find static files and templates when run from different working directories. The static and template directories are now resolved relative to the package location, and a clear error is raised if the static directory is missing. (68ce57, Eric Ma)

## Deprecations

- No deprecations in this release.

## Dependency Updates

- Updated FastAPI to 0.127.0, jaraco-functools to 4.4.0, and uvicorn to 0.40.0 for improved stability and compatibility. Also removed an obsolete migration issue file. (d38767, Eric Ma)
- Bumped rossjrw/pr-preview-action from 1.6.3 to 1.8.1 in CI workflows. (e698fc, dependabot[bot])
- Bumped prefix-dev/setup-pixi from 0.9.3 to 0.9.4 in CI workflows. (489d98, dependabot[bot])
